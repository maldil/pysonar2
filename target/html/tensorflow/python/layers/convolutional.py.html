  29 class Conv1D(keras_layers.Conv1D, base.Layer):
  30   """1D convolution layer (e.g. temporal convolution).
  31 
  32   This layer creates a convolution kernel that is convolved
  33   (actually cross-correlated) with the layer input to produce a tensor of
  34   outputs. If `use_bias` is True (and a `bias_initializer` is provided),
  35   a bias vector is created and added to the outputs. Finally, if
  36   `activation` is not `None`, it is applied to the outputs as well.
  37 
  38   Arguments:
  39     filters: Integer, the dimensionality of the output space (i.e. the number
  40       of filters in the convolution).
  41     kernel_size: An integer or tuple/list of a single integer, specifying the
  42       length of the 1D convolution window.
  43     strides: An integer or tuple/list of a single integer,
  44       specifying the stride length of the convolution.
  45       Specifying any stride value != 1 is incompatible with specifying
  46       any `dilation_rate` value != 1.
  47     padding: One of `"valid"` or `"same"` (case-insensitive).
  48     data_format: A string, one of `channels_last` (default) or `channels_first`.
  49       The ordering of the dimensions in the inputs.
  50       `channels_last` corresponds to inputs with shape
  51       `(batch, length, channels)` while `channels_first` corresponds to
  52       inputs with shape `(batch, channels, length)`.
  53     dilation_rate: An integer or tuple/list of a single integer, specifying
  54       the dilation rate to use for dilated convolution.
  55       Currently, specifying any `dilation_rate` value != 1 is
  56       incompatible with specifying any `strides` value != 1.
  57     activation: Activation function. Set it to None to maintain a
  58       linear activation.
  59     use_bias: Boolean, whether the layer uses a bias.
  60     kernel_initializer: An initializer for the convolution kernel.
  61     bias_initializer: An initializer for the bias vector. If None, the default
  62       initializer will be used.
  63     kernel_regularizer: Optional regularizer for the convolution kernel.
  64     bias_regularizer: Optional regularizer for the bias vector.
  65     activity_regularizer: Optional regularizer function for the output.
  66     kernel_constraint: Optional projection function to be applied to the
  67         kernel after being updated by an `Optimizer` (e.g. used to implement
  68         norm constraints or value constraints for layer weights). The function
  69         must take as input the unprojected variable and must return the
  70         projected variable (which must have the same shape). Constraints are
  71         not safe to use when doing asynchronous distributed training.
  72     bias_constraint: Optional projection function to be applied to the
  73         bias after being updated by an `Optimizer`.
  74     trainable: Boolean, if `True` also add variables to the graph collection
  75       `GraphKeys.TRAINABLE_VARIABLES` (see `tf.Variable`).
  76     name: A string, the name of the layer.
  77   """
