  42 dtype `self.dtype` and be in the `(self.event_shape() - 1)`-simplex, i.e.,
  43 `tf.reduce_sum(value, -1) = 1`. It must have a shape compatible with
  44 `self.batch_shape() + self.event_shape()`."""

  47 @tf_export("distributions.Dirichlet")
  48 class Dirichlet(distribution.Distribution):
  49   """Dirichlet distribution.
  50 
  51   The Dirichlet distribution is defined over the
  52   [`(k-1)`-simplex](https://en.wikipedia.org/wiki/Simplex) using a positive,
  53   length-`k` vector `concentration` (`k > 1`). The Dirichlet is identically the
  54   Beta distribution when `k = 2`.
  55 
  56   #### Mathematical Details
  57 
  58   The Dirichlet is a distribution over the open `(k-1)`-simplex, i.e.,
  59 
  60   ```none
  61   S^{k-1} = { (x_0, ..., x_{k-1}) in R^k : sum_j x_j = 1 and all_j x_j > 0 }.
  62   ```
  63 
  64   The probability density function (pdf) is,
  65 
  66   ```none
  67   pdf(x; alpha) = prod_j x_j**(alpha_j - 1) / Z
  68   Z = prod_j Gamma(alpha_j) / Gamma(sum_j alpha_j)
  69   ```
  70 
  71   where:
  72 
  73   * `x in S^{k-1}`, i.e., the `(k-1)`-simplex,
  74   * `concentration = alpha = [alpha_0, ..., alpha_{k-1}]`, `alpha_j > 0`,
  75   * `Z` is the normalization constant aka the [multivariate beta function](
  76     https://en.wikipedia.org/wiki/Beta_function#Multivariate_beta_function),
  77     and,
  78   * `Gamma` is the [gamma function](
  79     https://en.wikipedia.org/wiki/Gamma_function).
  80 
  81   The `concentration` represents mean total counts of class occurrence, i.e.,
  82 
  83   ```none
  84   concentration = alpha = mean * total_concentration
  85   ```
  86 
  87   where `mean` in `S^{k-1}` and `total_concentration` is a positive real number
  88   representing a mean total count.
  89 
  90   Distribution parameters are automatically broadcast in all functions; see
  91   examples for details.
  92 
  93   Warning: Some components of the samples can be zero due to finite precision.
  94   This happens more often when some of the concentrations are very small.
  95   Make sure to round the samples to `np.finfo(dtype).tiny` before computing the
  96   density.
  97 
  98   Samples of this distribution are reparameterized (pathwise differentiable).
  99   The derivatives are computed using the approach described in the paper
 100 
 101   [Michael Figurnov, Shakir Mohamed, Andriy Mnih.
 102   Implicit Reparameterization Gradients, 2018](https://arxiv.org/abs/1805.08498)
 103 
 104   #### Examples
 105 
 106   ```python
 107   import tensorflow_probability as tfp
 108   tfd = tfp.distributions
 109 
 110   # Create a single trivariate Dirichlet, with the 3rd class being three times
 111   # more frequent than the first. I.e., batch_shape=[], event_shape=[3].
 112   alpha = [1., 2, 3]
 113   dist = tfd.Dirichlet(alpha)
 114 
 115   dist.sample([4, 5])  # shape: [4, 5, 3]
 116 
 117   # x has one sample, one batch, three classes:
 118   x = [.2, .3, .5]   # shape: [3]
 119   dist.prob(x)       # shape: []
 120 
 121   # x has two samples from one batch:
 122   x = [[.1, .4, .5],
 123        [.2, .3, .5]]
 124   dist.prob(x)         # shape: [2]
 125 
 126   # alpha will be broadcast to shape [5, 7, 3] to match x.
 127   x = [[...]]   # shape: [5, 7, 3]
 128   dist.prob(x)  # shape: [5, 7]
 129   ```
 130 
 131   ```python
 132   # Create batch_shape=[2], event_shape=[3]:
 133   alpha = [[1., 2, 3],
 134            [4, 5, 6]]   # shape: [2, 3]
 135   dist = tfd.Dirichlet(alpha)
 136 
 137   dist.sample([4, 5])  # shape: [4, 5, 2, 3]
 138 
 139   x = [.2, .3, .5]
 140   # x will be broadcast as [[.2, .3, .5],
 141   #                         [.2, .3, .5]],
 142   # thus matching batch_shape [2, 3].
 143   dist.prob(x)         # shape: [2]
 144   ```
 145 
 146   Compute the gradients of samples w.r.t. the parameters:
 147 
 148   ```python
 149   alpha = tf.constant([1.0, 2.0, 3.0])
 150   dist = tfd.Dirichlet(alpha)
 151   samples = dist.sample(5)  # Shape [5, 3]
 152   loss = tf.reduce_mean(tf.square(samples))  # Arbitrary loss function
 153   # Unbiased stochastic gradients of the loss function
 154   grads = tf.gradients(loss, alpha)
 155   ```
 156 
 157   """
 158 
 159   def __init__(self,
 160                concentration,
 161                validate_args=False,
 162                allow_nan_stats=True,
 163                name="Dirichlet"):
 164     """Initialize a batch of Dirichlet distributions.
 165 
 166     Args:
 167       concentration: Positive floating-point `Tensor` indicating mean number
 168         of class occurrences; aka "alpha". Implies `self.dtype`, and
 169         `self.batch_shape`, `self.event_shape`, i.e., if
 170         `concentration.shape = [N1, N2, ..., Nm, k]` then
 171         `batch_shape = [N1, N2, ..., Nm]` and
 172         `event_shape = [k]`.
 173       validate_args: Python `bool`, default `False`. When `True` distribution
 174         parameters are checked for validity despite possibly degrading runtime
 175         performance. When `False` invalid inputs may silently render incorrect
 176         outputs.
 177       allow_nan_stats: Python `bool`, default `True`. When `True`, statistics
 178         (e.g., mean, mode, variance) use the value "`NaN`" to indicate the
 179         result is undefined. When `False`, an exception is raised if one or
 180         more of the statistic's batch members are undefined.
 181       name: Python `str` name prefixed to Ops created by this class.
 182     """
 183     parameters = dict(locals())
 184     with ops.name_scope(name, values=[concentration]) as name:
 185       self._concentration = self._maybe_assert_valid_concentration(
 186           ops.convert_to_tensor(concentration, name="concentration"),
 187           validate_args)
 188       self._total_concentration = math_ops.reduce_sum(self._concentration, -1)
 189     super(Dirichlet, self).__init__(
 190         dtype=self._concentration.dtype,
 191         validate_args=validate_args,
 192         allow_nan_stats=allow_nan_stats,
 193         reparameterization_type=distribution.FULLY_REPARAMETERIZED,
 194         parameters=parameters,
 195         graph_parents=[self._concentration,
 196                        self._total_concentration],
 197         name=name)
 198 
 199   @property
 200   def concentration(self):
 201     """Concentration parameter; expected counts for that coordinate."""
 202     return self._concentration
 203 
 204   @property
 205   def total_concentration(self):
 206     """Sum of last dim of concentration parameter."""
 207     return self._total_concentration
 208 
 209   def _batch_shape_tensor(self):
 210     return array_ops.shape(self.total_concentration)
 211 
 212   def _batch_shape(self):
 213     return self.total_concentration.get_shape()
 214 
 215   def _event_shape_tensor(self):
 216     return array_ops.shape(self.concentration)[-1:]
 217 
 218   def _event_shape(self):
 219     return self.concentration.get_shape().with_rank_at_least(1)[-1:]
 220 
 221   def _sample_n(self, n, seed=None):
 222     gamma_sample = random_ops.random_gamma(
 223         shape=[n],
 224         alpha=self.concentration,
 225         dtype=self.dtype,
 226         seed=seed)
 227     return gamma_sample / math_ops.reduce_sum(gamma_sample, -1, keepdims=True)
 228 
 229   @distribution_util.AppendDocstring(_dirichlet_sample_note)
 230   def _log_prob(self, x):
 231     return self._log_unnormalized_prob(x) - self._log_normalization()
 232 
 233   @distribution_util.AppendDocstring(_dirichlet_sample_note)
 234   def _prob(self, x):
 235     return math_ops.exp(self._log_prob(x))
 236 
 237   def _log_unnormalized_prob(self, x):
 238     x = self._maybe_assert_valid_sample(x)
 239     return math_ops.reduce_sum((self.concentration - 1.) * math_ops.log(x), -1)
 240 
 241   def _log_normalization(self):
 242     return special_math_ops.lbeta(self.concentration)
 243 
 244   def _entropy(self):
 245     k = math_ops.cast(self.event_shape_tensor()[0], self.dtype)
 246     return (
 247         self._log_normalization()
 248         + ((self.total_concentration - k)
 249            * math_ops.digamma(self.total_concentration))
 250         - math_ops.reduce_sum(
 251             (self.concentration - 1.) * math_ops.digamma(self.concentration),
 252             axis=-1))
 253 
 254   def _mean(self):
 255     return self.concentration / self.total_concentration[..., array_ops.newaxis]
 256 
 257   def _covariance(self):
 258     x = self._variance_scale_term() * self._mean()
 259     return array_ops.matrix_set_diag(
 260         -math_ops.matmul(x[..., array_ops.newaxis],
 261                          x[..., array_ops.newaxis, :]),  # outer prod
 262         self._variance())
 263 
 264   def _variance(self):
 265     scale = self._variance_scale_term()
 266     x = scale * self._mean()
 267     return x * (scale - x)
 268 
 269   def _variance_scale_term(self):
 270     """Helper to `_covariance` and `_variance` which computes a shared scale."""
 271     return math_ops.rsqrt(1. + self.total_concentration[..., array_ops.newaxis])
 272 
 273   @distribution_util.AppendDocstring(
 274       """Note: The mode is undefined when any `concentration <= 1`. If
 275       `self.allow_nan_stats` is `True`, `NaN` is used for undefined modes. If
 276       `self.allow_nan_stats` is `False` an exception is raised when one or more
 277       modes are undefined.""")
 278   def _mode(self):
 279     k = math_ops.cast(self.event_shape_tensor()[0], self.dtype)
 280     mode = (self.concentration - 1.) / (
 281         self.total_concentration[..., array_ops.newaxis] - k)
 282     if self.allow_nan_stats:
 283       nan = array_ops.fill(
 284           array_ops.shape(mode),
 285           np.array(np.nan, dtype=self.dtype.as_numpy_dtype()),
 286           name="nan")
 287       return array_ops.where(
 288           math_ops.reduce_all(self.concentration > 1., axis=-1),
 289           mode, nan)
 290     return control_flow_ops.with_dependencies([
 291         check_ops.assert_less(
 292             array_ops.ones([], self.dtype),
 293             self.concentration,
 294             message="Mode undefined when any concentration <= 1"),
 295     ], mode)
 296 
 297   def _maybe_assert_valid_concentration(self, concentration, validate_args):
 298     """Checks the validity of the concentration parameter."""
 299     if not validate_args:
 300       return concentration
 301     return control_flow_ops.with_dependencies([
 302         check_ops.assert_positive(
 303             concentration,
 304             message="Concentration parameter must be positive."),
 305         check_ops.assert_rank_at_least(
 306             concentration, 1,
 307             message="Concentration parameter must have >=1 dimensions."),
 308         check_ops.assert_less(
 309             1, array_ops.shape(concentration)[-1],
 310             message="Concentration parameter must have event_size >= 2."),
 311     ], concentration)
 312 
 313   def _maybe_assert_valid_sample(self, x):
 314     """Checks the validity of a sample."""
 315     if not self.validate_args:
 316       return x
 317     return control_flow_ops.with_dependencies([
 318         check_ops.assert_positive(x, message="samples must be positive"),
 319         check_ops.assert_near(
 320             array_ops.ones([], dtype=self.dtype),
 321             math_ops.reduce_sum(x, -1),
 322             message="sample last-dimension must sum to `1`"),
 323     ], x)

 326 @kullback_leibler.RegisterKL(Dirichlet, Dirichlet)
 327 def _kl_dirichlet_dirichlet(d1, d2, name=None):
 328   """Batchwise KL divergence KL(d1 || d2) with d1 and d2 Dirichlet.
 329 
 330   Args:
 331     d1: instance of a Dirichlet distribution object.
 332     d2: instance of a Dirichlet distribution object.
 333     name: (optional) Name to use for created operations.
 334       default is "kl_dirichlet_dirichlet".
 335 
 336   Returns:
 337     Batchwise KL(d1 || d2)
 338   """
 339   with ops.name_scope(name, "kl_dirichlet_dirichlet", values=[
 340       d1.concentration, d2.concentration]):
 341     # The KL between Dirichlet distributions can be derived as follows. We have
 342     #
 343     #   Dir(x; a) = 1 / B(a) * prod_i[x[i]^(a[i] - 1)]
 344     #
 345     # where B(a) is the multivariate Beta function:
 346     #
 347     #   B(a) = Gamma(a[1]) * ... * Gamma(a[n]) / Gamma(a[1] + ... + a[n])
 348     #
 349     # The KL is
 350     #
 351     #   KL(Dir(x; a), Dir(x; b)) = E_Dir(x; a){log(Dir(x; a) / Dir(x; b))}
 352     #
 353     # so we'll need to know the log density of the Dirichlet. This is
 354     #
 355     #   log(Dir(x; a)) = sum_i[(a[i] - 1) log(x[i])] - log B(a)
 356     #
 357     # The only term that matters for the expectations is the log(x[i]). To
 358     # compute the expectation of this term over the Dirichlet density, we can
 359     # use the following facts about the Dirichlet in exponential family form:
 360     #   1. log(x[i]) is a sufficient statistic
 361     #   2. expected sufficient statistics (of any exp family distribution) are
 362     #      equal to derivatives of the log normalizer with respect to
 363     #      corresponding natural parameters: E{T[i](x)} = dA/d(eta[i])
 364     #
 365     # To proceed, we can rewrite the Dirichlet density in exponential family
 366     # form as follows:
 367     #
 368     #   Dir(x; a) = exp{eta(a) . T(x) - A(a)}
 369     #
 370     # where '.' is the dot product of vectors eta and T, and A is a scalar:
 371     #
 372     #   eta[i](a) = a[i] - 1
 373     #     T[i](x) = log(x[i])
 374     #        A(a) = log B(a)
 375     #
 376     # Now, we can use fact (2) above to write
 377     #
 378     #   E_Dir(x; a)[log(x[i])]
 379     #       = dA(a) / da[i]
 380     #       = d/da[i] log B(a)
 381     #       = d/da[i] (sum_j lgamma(a[j])) - lgamma(sum_j a[j])
 382     #       = digamma(a[i])) - digamma(sum_j a[j])
 383     #
 384     # Putting it all together, we have
 385     #
 386     # KL[Dir(x; a) || Dir(x; b)]
 387     #     = E_Dir(x; a){log(Dir(x; a) / Dir(x; b)}
 388     #     = E_Dir(x; a){sum_i[(a[i] - b[i]) log(x[i])} - (lbeta(a) - lbeta(b))
 389     #     = sum_i[(a[i] - b[i]) * E_Dir(x; a){log(x[i])}] - lbeta(a) + lbeta(b)
 390     #     = sum_i[(a[i] - b[i]) * (digamma(a[i]) - digamma(sum_j a[j]))]
 391     #          - lbeta(a) + lbeta(b))
 392 
 393     digamma_sum_d1 = math_ops.digamma(
 394         math_ops.reduce_sum(d1.concentration, axis=-1, keepdims=True))
 395     digamma_diff = math_ops.digamma(d1.concentration) - digamma_sum_d1
 396     concentration_diff = d1.concentration - d2.concentration
 397 
 398     return (math_ops.reduce_sum(concentration_diff * digamma_diff, axis=-1) -
 399             special_math_ops.lbeta(d1.concentration) +
 400             special_math_ops.lbeta(d2.concentration))
