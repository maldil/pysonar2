  15 """Framework of debug wrapper sessions.
  16 
  17 A debug wrapper session is a wrapper around a TensorFlow Python Session.
  18 The wrapper preserves the Session interface, most importantly the run() method,
  19 while providing abilities to:
  20 a) Intercept a run() call to a wrapped session and insert debug tensor watches
  21    according to externally-specified debug URLs.
  22 
  23 b) Release control to an external (i.e., non-Session) object before and after
  24    the run() call, so that the external object can perform actions such as
  25    launching a UI to let users inspect the intermediate tensors and partition
  26    graphs from the run() call.
  27 
  28 c) (To be implemented) Intercept a run() call and give control to DebugStepper
  29    to let it perform stepping / continuing-to actions on the graph.
  30 
  31 b) (To be implemented in a future CL) Enter an instruction loop to let an
  32    external object (e.g., remote client) launch run() and cont() calls
  33    remotely.
  34 
  35 *** The lifetime of a debug wrapper session: ***
  36 
  37 1) The wrapper session is created by calling the constructor with a
  38    wrapped (normal) session as the argument:
  39      wrapper = FooDebugWrapperSession(sess)
  40    wherein FooDebugWrapperSession is a concrete subclass implementing the
  41    abstract BaseDebugWrapperSession class below.
  42 
  43 2) Near the end of the constructor call, the on_session_init() callback is
  44    invoked, with a OnSessionInitRequest object as the argument. The object
  45    carries the wrapped (normal) session object.
  46 
  47 3) The callback handles the request and returns a OnSessionInitResponse
  48    object with an action field, directing the wrapper session what to do next.
  49 
  50 If the action field in the OnSessionInitResponse is PROCEED, the constuctor
  51 returns. Control is released back to the caller of the constructor, which can
  52 invoke run() method of wrapper session with the same syntax as a non-wrapped
  53 session, e.g.,:
  54   wrapper.run(fetches, feed_dict=feeds, options=run_options)
  55 
  56 Below, A1 - A2 is the lifetime of a wrapper run() call if the action is
  57 PROCEED:
  58 
  59 A1) Right at the start of each run() call, the on_run_start() callback is
  60     invoked, with an OnRunStartRequest object carrying information such as
  61     the fetches, the feed dict, the run options and run metadata used in
  62     this run call, along with a count of how many run calls has occurred
  63     on this wrapper session. The callback then returns an OnRunStartResponse
  64     object, of which the action field directs what the wrapper session
  65     actually will do of the run() call.
  66 
  67     If the action is DEBUG_RUN, a debugged (tensor-watched) run will ensue,
  68     with the debug URLs supplied in the debug_urls field of the response.
  69     These can be file:// or grpc:// URLs, for example.
  70 
  71     If the action is NON_DEBUG_RUN, a non-debug (normal) run will ensue.
  72 
  73     If the action is INVOKE_STEPPER, no run() call will be issued to the
  74     wrapped session. But instead, a DebugStepper (i.e., "continuation
  75     debugger") will be used to perform stepping / continue-to actions on
  76     the graph.
  77 
  78 TODO(cais): The event loop for the DebugStepper will request additional
  79    callbacks including on_cont_start() and on_cont_end(). Add those.
  80 
  81 A2) Right before the run() returns, the on_run_end() callback is invoked,
  82     with an OnRunEndRequest object as the argument, which carries information
  83     including the actual action performed in the warpper run() call and the
  84     run_metadata from the run() call.
  85 
  86 However, if the action field in OnSessionInitResponse is
  87 REMOTE_INSTR_LOOP, the constructor will automatically invoke an instruction loop
  88 that gives the control to a remote caller.
  89 
  90 In the remote instruction loop, the following steps will happen:
  91 
  92 B1) Callback on_instr_start() is invoked. The callback will return an
  93     OnInstrStartResponse object with an action field which can order one of
  94     the following actions:
  95         i) a run() call with fetches, feeds and debug_urls specified.
  96        ii) a DebugStepper cont() call with target specified.
  97       iii) value overrides in the cached tensors from the DebugStepper.
  98        iv) exit the instruction loop.
  99 
 100 B2) The wrapper session carries out the action specified above.
 101 
 102 B3) If still in the instruction loop, the wrapper session invokes the
 103     on_instr_end() callback. After the on_instr_end() callback returns, jump
 104     back to B1.
 105 
 106 TODO(cais): Implemented the instruction loop in B1 - B3.
 107

 129 # Helper function.
 130 def _check_type(obj, expected_types):
 131   """Check if an object is of the expected type.
 132 
 133   Args:
 134     obj: The object being checked.
 135     expected_types: (`type` or an iterable of `type`s) The expected `type`(s)
 136       of obj.
 137 
 138   Raises:
 139       TypeError: If obj is not an instance of expected_type.
 140   """
 141   if not isinstance(obj, expected_types):
 142     raise TypeError("Expected type %s; got type %s" %
 143                     (expected_types, type(obj)))

 163 class OnSessionInitAction(object):
 164   """Enum-like values for possible action to take on session init."""
 165 
 166   # Proceed, without special actions, in the wrapper session initialization.
 167   # What action the wrapper session performs next is determined by the caller
 168   # of the wrapper session. E.g., it can call run().
 169   PROCEED = "proceed"
 170 
 171   # Instead of letting the caller of the wrapper session determine what actions
 172   # the wrapper session will perform next, enter a loop to receive instructions
 173   # from a remote client.
 174   # For example, TensorBoard visual debugger can use this action so that it can
 175   # launch session.run() calls remotely.
 176   REMOTE_INSTR_LOOP = "remote_instr_loop"

 192 class OnRunStartRequest(object):
 193   """Request to an on-run-start callback.
 194 
 195   This callback is invoked during a run() call of the debug-wrapper
 196   session, immediately after the run() call counter is incremented.
 197   """
 198 
 199   def __init__(self, fetches, feed_dict, run_options, run_metadata,
 200                run_call_count, is_callable_runner=False):
 201     """Constructor of `OnRunStartRequest`.
 202 
 203     Args:
 204       fetches: Fetch targets of the run() call.
 205       feed_dict: The feed dictionary to the run() call.
 206       run_options: RunOptions input to the run() call.
 207       run_metadata: RunMetadata input to the run() call.
 208         The above four arguments are identical to the input arguments to the
 209         run() method of a non-wrapped TensorFlow session.
 210       run_call_count: 1-based count of how many run calls (including this one)
 211         has been invoked.
 212       is_callable_runner: (bool) whether a runner returned by
 213         Session.make_callable is being run.
 214     """
 215     self.fetches = fetches
 216     self.feed_dict = feed_dict
 217     self.run_options = run_options
 218     self.run_metadata = run_metadata
 219     self.run_call_count = run_call_count
 220     self.is_callable_runner = is_callable_runner

 332 class BaseDebugWrapperSession(session.SessionInterface):
 333   """Base class of debug-wrapper session classes.
 334 
 335   Concrete classes that inherit from this class need to implement the abstract
 336   methods such as on_session_init, on_run_start and on_run_end.
 337   """
 338 
 339   # TODO(cais): Add on_cont_start and on_cont_end callbacks once the stepper is
 340   # is available.
 341 
 342   def __init__(self, sess, thread_name_filter=None,
 343                pass_through_operrors=False):
 344     """Constructor of `BaseDebugWrapperSession`.
 345 
 346     Args:
 347       sess: An (unwrapped) TensorFlow session instance. It should be a subtype
 348         of `BaseSession` or `tf.MonitoredSession`.
 349       thread_name_filter: Regular-expression filter (whitelist) for name(s) of
 350         thread(s) on which the wrapper session will be active. This regular
 351         expression is used in a start-anchored fashion on the thread name, i.e.,
 352         by applying the `match` method of the compiled pattern. The default
 353         `None` means that the wrapper session will be active on all threads.
 354         E.g., r"MainThread$", r"QueueRunnerThread.*".
 355       pass_through_operrors: If True, all captured OpErrors will be
 356         propagated.  By default this captures all OpErrors.
 357 
 358     Raises:
 359       ValueError: On invalid `OnSessionInitAction` value.
 360       NotImplementedError: If a non-DirectSession sess object is received.
 361     """
 362 
 363     _check_type(sess, (session.BaseSession, monitored_session.MonitoredSession))
 364 
 365     # The session being wrapped.
 366     self._sess = sess
 367     self._thread_name_filter_pattern = (re.compile(thread_name_filter)
 368                                         if thread_name_filter else None)
 369     # TODO(cais/kstevens): Unittest this pass through feature.
 370     self._pass_through_operrors = pass_through_operrors
 371 
 372     # Keeps track of number of run calls that have been performed on this
 373     # debug-wrapper session. The count can be used for purposes such as
 374     # displaying the state of the Session in a UI and determining a run
 375     # number-dependent debug URL.
 376     self._run_call_count = 0
 377 
 378     # Invoke on-session-init callback.
 379     response = self.on_session_init(OnSessionInitRequest(self._sess))
 380     _check_type(response, OnSessionInitResponse)
 381 
 382     if response.action == OnSessionInitAction.PROCEED:
 383       pass
 384     elif response.action == OnSessionInitAction.REMOTE_INSTR_LOOP:
 385       # TODO(cais): Implement REMOTE_INSTR_LOOP
 386       raise NotImplementedError(
 387           "OnSessionInitAction REMOTE_INSTR_LOOP has not been "
 388           "implemented.")
 389     else:
 390       raise ValueError(
 391           "Invalid OnSessionInitAction value: %s" % response.action)
 392 
 393     self._default_session_context_manager = None
 394 
 395     # A cache for callables created from CallableOptions.
 396     self._cached_callables_from_options = dict()
 397 
 398   @property
 399   def graph(self):
 400     return self._sess.graph
 401 
 402   @property
 403   def graph_def(self):
 404     return self._sess.graph_def
 405 
 406   @property
 407   def sess_str(self):
 408     return self._sess.sess_str
 409 
 410   @property
 411   def session(self):
 412     return self._sess
 413 
 414   def run(self,
 415           fetches,
 416           feed_dict=None,
 417           options=None,
 418           run_metadata=None,
 419           callable_runner=None,
 420           callable_runner_args=None,
 421           callable_options=None):
 422     """Wrapper around Session.run() that inserts tensor watch options.
 423 
 424     Args:
 425       fetches: Same as the `fetches` arg to regular `Session.run()`.
 426       feed_dict: Same as the `feed_dict` arg to regular `Session.run()`.
 427       options: Same as the `options` arg to regular `Session.run()`.
 428       run_metadata: Same as the `run_metadata` arg to regular `Session.run()`.
 429       callable_runner: A `callable` returned by `Session.make_callable()`.
 430         If not `None`, `fetches` and `feed_dict` must both be `None`.
 431         Mutually exclusive with `callable_options`.
 432       callable_runner_args: An optional list of arguments to `callable_runner`
 433         or for `callable_options`.
 434       callable_options: An instance of `config_pb2.CallableOptions`, to be
 435         used with `Session._make_callable_from_options()`. Mutually exclusive
 436         with `callable_runner`.
 437 
 438     Returns:
 439       Simply forwards the output of the wrapped `Session.run()` call.
 440 
 441     Raises:
 442       ValueError: On invalid `OnRunStartAction` value. Or if `callable_runner`
 443         is not `None` and either or both of `fetches` and `feed_dict` is `None`.
 444     """
 445     if callable_runner and callable_options:
 446       raise ValueError(
 447           "callable_runner and callable_options are mutually exclusive, but "
 448           "are both specified in this call to BaseDebugWrapperSession.run().")
 449 
 450     if callable_runner and (fetches or feed_dict):
 451       raise ValueError(
 452           "callable_runner and fetches/feed_dict are mutually exclusive, "
 453           "but are used simultaneously.")
 454     elif callable_options and (fetches or feed_dict):
 455       raise ValueError(
 456           "callable_options and fetches/feed_dict are mutually exclusive, "
 457           "but are used simultaneously.")
 458 
 459     self.increment_run_call_count()
 460     empty_fetches = not nest.flatten(fetches)
 461     if empty_fetches:
 462       tf_logging.info(
 463           "Due to empty fetches, tfdbg Session wrapper is letting a "
 464           "Session.run pass through without any debugging actions.")
 465     if self._is_disabled_thread() or empty_fetches:
 466       if callable_runner:
 467         return callable_runner(*callable_runner_args)
 468       elif callable_options:
 469         # pylint:disable=protected-access
 470         return self._sess._make_callable_from_options(
 471             callable_options)(*callable_runner_args)
 472         # pylint:enable=protected-access
 473       else:
 474         return self._sess.run(fetches,
 475                               feed_dict=feed_dict,
 476                               options=options,
 477                               run_metadata=run_metadata)
 478 
 479     # Invoke on-run-start callback and obtain response.
 480     run_start_resp = self.on_run_start(
 481         OnRunStartRequest(fetches, feed_dict, options, run_metadata,
 482                           self._run_call_count,
 483                           is_callable_runner=bool(callable_runner)))
 484     _check_type(run_start_resp, OnRunStartResponse)
 485 
 486     if run_start_resp.action == OnRunStartAction.DEBUG_RUN:
 487       # Decorate RunOption to fill in debugger tensor watch specifications.
 488       decorated_run_options = None
 489       if callable_options:
 490         callable_options_id = id(callable_options)
 491         if callable_options_id not in self._cached_callables_from_options:
 492           # Make a copy of callable_options to avoid mutating it.
 493           new_callable_options = config_pb2.CallableOptions()
 494           new_callable_options.CopyFrom(callable_options)
 495           decorated_run_options = new_callable_options.run_options
 496       else:
 497         decorated_run_options = options or config_pb2.RunOptions()
 498 
 499       run_metadata = run_metadata or config_pb2.RunMetadata()
 500 
 501       if decorated_run_options:
 502         self._decorate_run_options_for_debug(
 503             decorated_run_options,
 504             run_start_resp.debug_urls,
 505             debug_ops=run_start_resp.debug_ops,
 506             node_name_regex_whitelist=run_start_resp.node_name_regex_whitelist,
 507             op_type_regex_whitelist=run_start_resp.op_type_regex_whitelist,
 508             tensor_dtype_regex_whitelist=(
 509                 run_start_resp.tensor_dtype_regex_whitelist),
 510             tolerate_debug_op_creation_failures=(
 511                 run_start_resp.tolerate_debug_op_creation_failures))
 512 
 513       # Invoke the run() method of the wrapped Session. Catch any TensorFlow
 514       # runtime errors.
 515       tf_error = None
 516       try:
 517         if callable_runner:
 518           retvals = callable_runner(*callable_runner_args,
 519                                     options=decorated_run_options,
 520                                     run_metadata=run_metadata)
 521         elif callable_options:
 522           # pylint:disable=protected-access
 523           if callable_options_id in self._cached_callables_from_options:
 524             callable_object = self._cached_callables_from_options[
 525                 callable_options_id]
 526           else:
 527             callable_object = self._sess._make_callable_from_options(
 528                 new_callable_options)
 529             self._cached_callables_from_options[
 530                 callable_options_id] = callable_object
 531           # pylint:enable=protected-access
 532           retvals = callable_object(
 533               *callable_runner_args, run_metadata=run_metadata)
 534         else:
 535           retvals = self._sess.run(fetches,
 536                                    feed_dict=feed_dict,
 537                                    options=decorated_run_options,
 538                                    run_metadata=run_metadata)
 539       except errors.OpError as op_error:
 540         if self._pass_through_operrors:
 541           raise op_error
 542         tf_error = op_error
 543         retvals = op_error
 544 
 545       run_end_req = OnRunEndRequest(
 546           run_start_resp.action,
 547           run_metadata=run_metadata,
 548           client_graph_def=self._sess.graph.as_graph_def(),
 549           tf_error=tf_error)
 550 
 551     elif run_start_resp.action == OnRunStartAction.PROFILE_RUN:
 552       decorated_run_options = options or config_pb2.RunOptions()
 553       run_metadata = run_metadata or config_pb2.RunMetadata()
 554       self._decorate_run_options_for_profile(decorated_run_options)
 555       if callable_runner:
 556         retvals = callable_runner(*callable_runner_args,
 557                                   options=decorated_run_options,
 558                                   run_metadata=run_metadata)
 559       else:
 560         retvals = self._sess.run(fetches,
 561                                  feed_dict=feed_dict,
 562                                  options=decorated_run_options,
 563                                  run_metadata=run_metadata)
 564       run_end_req = OnRunEndRequest(
 565           run_start_resp.action,
 566           run_metadata=run_metadata,
 567           client_graph_def=self._sess.graph.as_graph_def())
 568     elif (run_start_resp.action == OnRunStartAction.NON_DEBUG_RUN or
 569           run_start_resp.action == OnRunStartAction.INVOKE_STEPPER):
 570       if callable_runner:
 571         raise NotImplementedError(
 572             "Stepper mode is not implemented for callables created by "
 573             "Session.make_callable().")
 574 
 575       if run_start_resp.action == OnRunStartAction.INVOKE_STEPPER:
 576         with stepper.NodeStepper(
 577             self._sess, fetches, feed_dict) as node_stepper:
 578           retvals = self.invoke_node_stepper(
 579               node_stepper, restore_variable_values_on_exit=True)
 580 
 581       # Invoke run() method of the wrapped session.
 582       retvals = self._sess.run(
 583           fetches,
 584           feed_dict=feed_dict,
 585           options=options,
 586           run_metadata=run_metadata)
 587 
 588       # Prepare arg for the on-run-end callback.
 589       run_end_req = OnRunEndRequest(run_start_resp.action)
 590     else:
 591       raise ValueError(
 592           "Invalid OnRunStartAction value: %s" % run_start_resp.action)
 593 
 594     # Invoke on-run-end callback and obtain response.
 595     run_end_resp = self.on_run_end(run_end_req)
 596     _check_type(run_end_resp, OnRunEndResponse)
 597     # Currently run_end_resp is only a placeholder. No action is taken on it.
 598 
 599     return retvals

 818 class WatchOptions(object):
 819   """Type for return values of watch_fn."""
 820 
 821   def __init__(self,
 822                debug_ops=None,
 823                node_name_regex_whitelist=None,
 824                op_type_regex_whitelist=None,
 825                tensor_dtype_regex_whitelist=None,
 826                tolerate_debug_op_creation_failures=False):
 827     """Constructor of WatchOptions: Debug watch options.
 828 
 829     Used as return values of `watch_fn`s.
 830 
 831     Args:
 832       debug_ops: (`str` or `list of str`) Debug ops to be used.
 833       node_name_regex_whitelist: Regular-expression whitelist for node_name,
 834         e.g., `"(weight_[0-9]+|bias_.*)"`
 835       op_type_regex_whitelist: Regular-expression whitelist for the op type of
 836         nodes, e.g., `"(Variable|Add)"`.
 837         If both `node_name_regex_whitelist` and `op_type_regex_whitelist`
 838         are set, the two filtering operations will occur in a logical `AND`
 839         relation. In other words, a node will be included if and only if it
 840         hits both whitelists.
 841       tensor_dtype_regex_whitelist: Regular-expression whitelist for Tensor
 842         data type, e.g., `"^int.*"`.
 843         This whitelist operates in logical `AND` relations to the two whitelists
 844         above.
 845       tolerate_debug_op_creation_failures: (`bool`) whether debug op creation
 846         failures (e.g., due to dtype incompatibility) are to be tolerated by not
 847         throwing exceptions.
 848     """
 849     if debug_ops:
 850       self.debug_ops = debug_ops
 851     else:
 852       self.debug_ops = ["DebugIdentity"]
 853     self.node_name_regex_whitelist = node_name_regex_whitelist
 854     self.op_type_regex_whitelist = op_type_regex_whitelist
 855     self.tensor_dtype_regex_whitelist = tensor_dtype_regex_whitelist
 856     self.tolerate_debug_op_creation_failures = (
 857         tolerate_debug_op_creation_failures)
 858 
 859   def __repr__(self):
 860     return ("WatchOptions(debug_ops=%r, node_name_regex_whitelist=%r, "
 861             "op_type_regex_whitelist=%r, tensor_dtype_regex_whitelist=%r, "
 862             "tolerate_debug_op_creation_failures=%r)" % (
 863                 self.debug_ops, self.node_name_regex_whitelist,
 864                 self.op_type_regex_whitelist, self.tensor_dtype_regex_whitelist,
 865                 self.tolerate_debug_op_creation_failures))

 871   def __init__(self, sess, watch_fn=None, thread_name_filter=None,
 872                pass_through_operrors=False):
 873     """Constructor of NonInteractiveDebugWrapperSession.
 874 
 875     Args:
 876       sess: The TensorFlow `Session` object being wrapped.
 877       watch_fn: (`Callable`) A Callable that maps the fetches and feeds of a
 878         debugged `Session.run()` call to `WatchOptions.`
 879         * Args:
 880           * `fetches`: the fetches to the `Session.run()` call.
 881           * `feeds`: the feeds to the `Session.run()` call.
 882 
 883         * Returns:
 884          (`tf_debug.WatchOptions`) An object containing debug options including
 885            the debug ops to use, the node names, op types and/or tensor data
 886            types to watch, etc. See the documentation of `tf_debug.WatchOptions`
 887            for more details.
 888       thread_name_filter: Regular-expression white list for threads on which the
 889         wrapper session will be active. See doc of `BaseDebugWrapperSession` for
 890         more details.
 891       pass_through_operrors: If true, all captured OpErrors will be
 892         propagated.  By default this captures all OpErrors.
 893     Raises:
 894        TypeError: If a non-None `watch_fn` is specified and it is not callable.
 895     """
 896 
 897     BaseDebugWrapperSession.__init__(
 898         self, sess, thread_name_filter=thread_name_filter,
 899         pass_through_operrors=pass_through_operrors)
 900 
 901     self._watch_fn = None
 902     if watch_fn is not None:
 903       if not callable(watch_fn):
 904         raise TypeError("watch_fn is not callable")
 905       self._watch_fn = watch_fn

 931         request.fetches, request.feed_dict)

 943   def _prepare_run_watch_config(self, fetches, feed_dict):
 944     """Get the debug_urls, and node/op whitelists for the current run() call.
 945 
 946     Args:
 947       fetches: Same as the `fetches` argument to `Session.run()`.
 948       feed_dict: Same as the `feed_dict argument` to `Session.run()`.
 949 
 950     Returns:
 951       debug_urls: (str or list of str) Debug URLs for the current run() call.
 952         Currently, the list consists of only one URL that is a file:// URL.
 953       watch_options: (WatchOptions) The return value of a watch_fn, containing
 954         options including debug_ops, and whitelists.
 955     """
 956 
 957     debug_urls = self.prepare_run_debug_urls(fetches, feed_dict)
 958     if self._watch_fn is None:
 959       watch_options = WatchOptions()
 960     else:
 961       watch_options = self._watch_fn(fetches, feed_dict)
 962       if isinstance(watch_options, tuple):
 963         # For legacy return type (tuples).
 964         watch_options = WatchOptions(*watch_options)
 965 
 966     return debug_urls, watch_options

 969     """See doc of BaseDebugWrapperSession.on_run_end."""
