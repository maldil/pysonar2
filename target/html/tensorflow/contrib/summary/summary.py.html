  15 """TensorFlow Summary API v2.
  16 
  17 The operations in this package are safe to use with eager execution turned on or
  18 off. It has a more flexible API that allows summaries to be written directly
  19 from ops to places other than event log files, rather than propagating protos
  20 from `tf.summary.merge_all` to `tf.summary.FileWriter`.
  21 
  22 To use with eager execution enabled, write your code as follows:
  23 
  24 ```python
  25 global_step = tf.train.get_or_create_global_step()
  26 summary_writer = tf.contrib.summary.create_file_writer(
  27     train_dir, flush_millis=10000)
  28 with summary_writer.as_default(), tf.contrib.summary.always_record_summaries():
  29   # model code goes here
  30   # and in it call
  31   tf.contrib.summary.scalar("loss", my_loss)
  32   # In this case every call to tf.contrib.summary.scalar will generate a record
  33   # ...
  34 ```
  35 
  36 To use it with graph execution, write your code as follows:
  37 
  38 ```python
  39 global_step = tf.train.get_or_create_global_step()
  40 summary_writer = tf.contrib.summary.create_file_writer(
  41     train_dir, flush_millis=10000)
  42 with summary_writer.as_default(), tf.contrib.summary.always_record_summaries():
  43   # model definition code goes here
  44   # and in it call
  45   tf.contrib.summary.scalar("loss", my_loss)
  46   # In this case every call to tf.contrib.summary.scalar will generate an op,
  47   # note the need to run tf.contrib.summary.all_summary_ops() to make sure these
  48   # ops get executed.
  49   # ...
  50   train_op = ....
  51 
  52 with tf.Session(...) as sess:
  53   tf.global_variables_initializer().run()
  54   tf.contrib.summary.initialize(graph=tf.get_default_graph())
  55   # ...
  56   while not_done_training:
  57     sess.run([train_op, tf.contrib.summary.all_summary_ops()])
  58     # ...
  59 ```
  60 """
