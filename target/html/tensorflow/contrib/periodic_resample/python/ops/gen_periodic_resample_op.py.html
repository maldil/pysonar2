  31 
  32   This function implements a slightly more generic version of the subpixel
  33   convolutions found in this [paper](https://arxiv.org/abs/1609.05158).
  34 
  35   The formula for computing the elements in the `output` tensor is as follows:
  36 
  37     `T` = `values` tensor of rank `R`
  38 
  39     `S` = desired `shape` of output tensor (vector of length `R`)
  40 
  41     `P` = `output` tensor of rank `R`
  42 
  43     \\((T_1,\\ldots,T_R)\\) = shape(`T`)
  44 
  45     \\([S_1,\\ldots,S_q,\\ldots,S_R]\\) = elements of vector `S`
  46 
  47     A single element in `S` is left unspecified (denoted \\(S_q=-1\\)).
  48 
  49     Let \\(f_i\\) denote the (possibly non-integer) factor that relates the original
  50     dimension to the desired dimensions, \\(S_i=f_i T_i\\), for \\(i\\neq q\\) where
  51     \\(f_i>0\\).
  52 
  53     Define the following:
  54 
  55     \\(g_i=\\lceil f_i\\rceil\\)
  56 
  57     \\(t=\\prod_i T_i\\)
  58 
  59     \\(s=\\prod_{i\\neq q} S_i\\)
  60 
  61     \\(S_q\\) can then be defined by \\(S_q=\\lfloor t/s\\rfloor\\).
  62     The elements of the resulting tensor are defined as
  63 
  64     \\(P_{s_1,\\ldots,s_R}=T_{h_1,\\ldots,h_q,\\ldots,h_R}\\).
  65 
  66     The \\(h_i\\) (\\(i\\neq q\\)) are defined by \\(h_i=\\lfloor s_i/g_i\\rfloor\\).
  67 
  68     \\(h_q=S_q\\sum_{j\\neq q}^{q-1}G_j \\mathrm{mod}(s_j,g_j) + s_q\\), where
  69     \\(G_j=\\prod_{i}^{j-1}g_i\\) (\\(G_0=1\\)).

  71   One drawback of this method is that whenever the output dimensions are slightly
  72   less than integer multiples of the input dimensions, many of the tensor elements
  73   are repeated in an inefficient way. This is resolved by specifying that all
  74   desired dimensions are integer multiples of the input tensor.
  75

  79   `input` is [[ 0  1  2  3]

  81               [ 8  9 10 11]]

  84                                               [ 2  3]
  85                                               [ 4  5]
  86                                               [ 6  7]
  87                                               [ 8  9]
  88                                               [10 11]]
  89   ```
