 655 
 656     node.
 657

 663      the logistic regression model that translates from node features to
 664      probabilities.
 665     tree_biases: `tree_biases[i]` gives the bias of the logistic
 666      regression model that translates from node features to
 667      probabilities.
 668     path_probability: `path_probability[i]` gives the probability of reaching each
 669      node in `path[i]`.
 670     path: `path[i][j]` gives the jth node in the path taken by the ith data
