  16 """An estimator is a rule for calculating an estimate of a given quantity (deprecated).
  17 
  18 These classes are deprecated and replaced with `tf.estimator`.
  19 
  20 See [contrib/learn/README.md](https://www.tensorflow.org/code/tensorflow/contrib/learn/README.md)
  21 for migration instructions.
  22 
  23 # Estimators
  24 
  25 * **Estimators** are used to train and evaluate TensorFlow models.
  26 They support regression and classification problems.
  27 * **Classifiers** are functions that have discrete outcomes.
  28 * **Regressors** are functions that predict continuous values.
  29 
  30 ## Choosing the correct estimator
  31 
  32 * For **Regression** problems use one of the following:
  33     * `LinearRegressor`: Uses linear model.
  34     * `DNNRegressor`: Uses DNN.
  35     * `DNNLinearCombinedRegressor`: Uses Wide & Deep.
  36     * `TensorForestEstimator`: Uses RandomForest.
  37       See tf.contrib.tensor_forest.client.random_forest.TensorForestEstimator.
  38     * `Estimator`: Use when you need a custom model.
  39 
  40 * For **Classification** problems use one of the following:
  41     * `LinearClassifier`: Multiclass classifier using Linear model.
  42     * `DNNClassifier`: Multiclass classifier using DNN.
  43     * `DNNLinearCombinedClassifier`: Multiclass classifier using Wide & Deep.
  44     * `TensorForestEstimator`: Uses RandomForest.
  45       See tf.contrib.tensor_forest.client.random_forest.TensorForestEstimator.
  46     * `SVM`: Binary classifier using linear SVMs.
  47     * `LogisticRegressor`: Use when you need custom model for binary
  48        classification.
  49     * `Estimator`: Use when you need custom model for N class classification.
  50 
  51 ## Pre-canned Estimators
  52 
  53 Pre-canned estimators are machine learning estimators premade for general
  54 purpose problems. If you need more customization, you can always write your
  55 own custom estimator as described in the section below.
  56 
  57 Pre-canned estimators are tested and optimized for speed and quality.
  58 
  59 ### Define the feature columns
  60 
  61 Here are some possible types of feature columns used as inputs to a pre-canned
  62 estimator.
  63 
  64 Feature columns may vary based on the estimator used. So you can see which
  65 feature columns are fed to each estimator in the below section.
  66 
  67 ```python
  68 sparse_feature_a = sparse_column_with_keys(
  69     column_name="sparse_feature_a", keys=["AB", "CD", ...])
  70 
  71 embedding_feature_a = embedding_column(
  72     sparse_id_column=sparse_feature_a, dimension=3, combiner="sum")
  73 
  74 sparse_feature_b = sparse_column_with_hash_bucket(
  75     column_name="sparse_feature_b", hash_bucket_size=1000)
  76 
  77 embedding_feature_b = embedding_column(
  78     sparse_id_column=sparse_feature_b, dimension=16, combiner="sum")
  79 
  80 crossed_feature_a_x_b = crossed_column(
  81     columns=[sparse_feature_a, sparse_feature_b], hash_bucket_size=10000)
  82 
  83 real_feature = real_valued_column("real_feature")
  84 real_feature_buckets = bucketized_column(
  85     source_column=real_feature,
  86     boundaries=[18, 25, 30, 35, 40, 45, 50, 55, 60, 65])
  87 ```
  88 
  89 ### Create the pre-canned estimator
  90 
  91 DNNClassifier, DNNRegressor, and DNNLinearCombinedClassifier are all pretty
  92 similar to each other in how you use them. You can easily plug in an
  93 optimizer and/or regularization to those estimators.
  94 
  95 #### DNNClassifier
  96 
  97 A classifier for TensorFlow DNN models.
  98 
  99 ```python
 100 my_features = [embedding_feature_a, embedding_feature_b]
 101 estimator = DNNClassifier(
 102     feature_columns=my_features,
 103     hidden_units=[1024, 512, 256],
 104     optimizer=tf.train.ProximalAdagradOptimizer(
 105         learning_rate=0.1,
 106         l1_regularization_strength=0.001
 107     ))
 108 ```
 109 
 110 #### DNNRegressor
 111 
 112 A regressor for TensorFlow DNN models.
 113 
 114 ```python
 115 my_features = [embedding_feature_a, embedding_feature_b]
 116 
 117 estimator = DNNRegressor(
 118 feature_columns=my_features,
 119 hidden_units=[1024, 512, 256])
 120 
 121 # Or estimator using the ProximalAdagradOptimizer optimizer with
 122 # regularization.
 123 estimator = DNNRegressor(
 124     feature_columns=my_features,
 125     hidden_units=[1024, 512, 256],
 126     optimizer=tf.train.ProximalAdagradOptimizer(
 127         learning_rate=0.1,
 128         l1_regularization_strength=0.001
 129     ))
 130 ```
 131 
 132 #### DNNLinearCombinedClassifier
 133 
 134 A classifier for TensorFlow Linear and DNN joined training models.
 135 
 136 * Wide and deep model
 137 * Multi class (2 by default)
 138 
 139 ```python
 140 my_linear_features = [crossed_feature_a_x_b]
 141 my_deep_features = [embedding_feature_a, embedding_feature_b]
 142 estimator = DNNLinearCombinedClassifier(
 143       # Common settings
 144       n_classes=n_classes,
 145       weight_column_name=weight_column_name,
 146       # Wide settings
 147       linear_feature_columns=my_linear_features,
 148       linear_optimizer=tf.train.FtrlOptimizer(...),
 149       # Deep settings
 150       dnn_feature_columns=my_deep_features,
 151       dnn_hidden_units=[1000, 500, 100],
 152       dnn_optimizer=tf.train.AdagradOptimizer(...))
 153 ```
 154 
 155 #### LinearClassifier
 156 
 157 Train a linear model to classify instances into one of multiple possible
 158 classes. When number of possible classes is 2, this is binary classification.
 159 
 160 ```python
 161 my_features = [sparse_feature_b, crossed_feature_a_x_b]
 162 estimator = LinearClassifier(
 163     feature_columns=my_features,
 164     optimizer=tf.train.FtrlOptimizer(
 165         learning_rate=0.1,
 166         l1_regularization_strength=0.001
 167     ))
 168 ```
 169 
 170 #### LinearRegressor
 171 
 172 Train a linear regression model to predict a label value given observation of
 173 feature values.
 174 
 175 ```python
 176 my_features = [sparse_feature_b, crossed_feature_a_x_b]
 177 estimator = LinearRegressor(
 178     feature_columns=my_features)
 179 ```
 180 
 181 ### LogisticRegressor
 182 
 183 Logistic regression estimator for binary classification.
 184 
 185 ```python
 186 # See tf.contrib.learn.Estimator(...) for details on model_fn structure
 187 def my_model_fn(...):
 188   pass
 189 
 190 estimator = LogisticRegressor(model_fn=my_model_fn)
 191 
 192 # Input builders
 193 def input_fn_train:
 194   pass
 195 
 196 estimator.fit(input_fn=input_fn_train)
 197 estimator.predict(x=x)
 198 ```
 199 
 200 #### SVM - Support Vector Machine
 201 
 202 Support Vector Machine (SVM) model for binary classification.
 203 
 204 Currently only linear SVMs are supported.
 205 
 206 ```python
 207 my_features = [real_feature, sparse_feature_a]
 208 estimator = SVM(
 209     example_id_column='example_id',
 210     feature_columns=my_features,
 211     l2_regularization=10.0)
 212 ```
 213 
 214 #### DynamicRnnEstimator
 215 
 216 An `Estimator` that uses a recurrent neural network with dynamic unrolling.
 217 
 218 ```python
 219 problem_type = ProblemType.CLASSIFICATION  # or REGRESSION
 220 prediction_type = PredictionType.SINGLE_VALUE  # or MULTIPLE_VALUE
 221 
 222 estimator = DynamicRnnEstimator(problem_type,
 223                                 prediction_type,
 224                                 my_feature_columns)
 225 ```
 226 
 227 ### Use the estimator
 228 
 229 There are two main functions for using estimators, one of which is for
 230 training, and one of which is for evaluation.
 231 You can specify different data sources for each one in order to use different
 232 datasets for train and eval.
 233 
 234 ```python
 235 # Input builders
 236 def input_fn_train: # returns x, Y
 237   ...
 238 estimator.fit(input_fn=input_fn_train)
 239 
 240 def input_fn_eval: # returns x, Y
 241   ...
 242 estimator.evaluate(input_fn=input_fn_eval)
 243 estimator.predict(x=x)
 244 ```
 245 
 246 ## Creating Custom Estimator
 247 
 248 To create a custom `Estimator`, provide a function to `Estimator`'s
 249 constructor that builds your model (`model_fn`, below):
 250 
 251 
 252 ```python
 253 estimator = tf.contrib.learn.Estimator(
 254       model_fn=model_fn,
 255       model_dir=model_dir)  # Where the model's data (e.g., checkpoints)
 256                             # are saved.
 257 ```
 258 
 259 Here is a skeleton of this function, with descriptions of its arguments and
 260 return values in the accompanying tables:
 261 
 262 ```python
 263 def model_fn(features, targets, mode, params):
 264    # Logic to do the following:
 265    # 1. Configure the model via TensorFlow operations
 266    # 2. Define the loss function for training/evaluation
 267    # 3. Define the training operation/optimizer
 268    # 4. Generate predictions
 269    return predictions, loss, train_op
 270 ```
 271 
 272 You may use `mode` and check against
 273 `tf.contrib.learn.ModeKeys.{TRAIN, EVAL, INFER}` to parameterize `model_fn`.
 274 
 275 In the Further Reading section below, there is an end-to-end TensorFlow
 276 tutorial for building a custom estimator.
