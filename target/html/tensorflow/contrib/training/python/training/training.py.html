  15 """Contains various routines and helper functions for training models.
  16 
  17 This script contains various functions for training models. These include
  18 manipulating gradients, creating a `train_op` (an operation that computes the
  19 loss and applies the gradients) and a training loop function. The training loop
  20 allows the user to pass in the `train_op` and runs the optimization according
  21 to user-specified arguments.
  22 
  23 ************************************
  24 * A simple working training script *
  25 ************************************
  26 
  27   # Load data and create the model:
  28   images, labels = LoadData(...)
  29   predictions = MyModel(images)
  30 
  31   # Define the loss:
  32   tf.contrib.losses.log_loss(predictions, labels)
  33   total_loss = tf.contrib.losses.get_total_loss()
  34 
  35   # Define the optimizer:
  36   optimizer = tf.train.MomentumOptimizer(FLAGS.learning_rate, FLAGS.momentum)
  37 
  38   # Create the train_op
  39   train_op = tf.contrib.training.create_train_op(total_loss, optimizer)
  40 
  41   # Run training.
  42   tf.contrib.training.train(train_op, my_log_dir)
  43 
  44 *************************
  45 * Creating the train_op *
  46 *************************
  47 
  48 In order to use the `train` function, one needs a train_op: an `Operation` that
  49 (a) computes the loss, (b) applies the gradients to update the weights and
  50 (c) returns the value of the loss. tf.contrib.training.create_train_op creates
  51 such an `Operation`. This function also provides the ability to manipulate
  52 the gradients using a few arguments:
  53 
  54   # Create the train_op and clip the gradient norms:
  55   train_op = tf.contrib.training.create_train_op(
  56       total_loss,
  57       optimizer,
  58       transform_grads_fn=clip_gradient_norms_fn(3))
  59 
  60   # Create the train_op and scale the gradients by providing a map from variable
  61   # name (or variable) to a scaling coefficient:
  62   def transform_grads_fn(grads):
  63     gradient_multipliers = {
  64       'conv0/weights': 1.2,
  65       'fc8/weights': 3.4,
  66     }
  67     return tf.contrib.training.multiply_gradients(
  68             grads, gradient_multipliers)
  69 
  70   train_op = tf.contrib.training.create_train_op(
  71       total_loss,
  72       optimizer,
  73       transform_grads_fn=transform_grads_fn)
  74 
  75 ****************************************************************
  76 * Performing additional (non-gradient) updates during training *
  77 ****************************************************************
  78 
  79 Many networks utilize modules, like BatchNorm, that require performing a series
  80 of non-gradient updates during training. tf.contrib.training.create_train_op
  81 allows a user to pass in a list of update_ops to call along with the gradient
  82 updates.
  83 
  84   train_op = tf.contrib.training.create_train_op(
  85       total_loss, optimizer, update_ops)
  86 
  87 By default, tf.contrib.training.create_train_op includes all update ops that are
  88 part of the `tf.GraphKeys.UPDATE_OPS` collection. Additionally, the
  89 tf.contrib.layers.batch_norm function adds the moving mean and moving variance
  90 updates to this collection. Consequently, users who want to use
  91 tf.contrib.layers.batch_norm will not need to take any additional steps in order
  92 to have the moving mean and moving variance updates be computed.
  93 
  94 However, users with additional, specialized updates can either override the
  95 default update ops or simply add additional update ops to the
  96 `tf.GraphKeys.UPDATE_OPS` collection:
  97 
  98   # Force `create_train_op` to NOT use ANY update_ops:
  99   train_op = tf.contrib.training.create_train_op(
 100      total_loss,
 101      optimizer,
 102      update_ops=[])
 103 
 104   # Use an alternative set of update ops:
 105   train_op = tf.contrib.training.create_train_op(
 106      total_loss,
 107      optimizer,
 108      update_ops=my_other_update_ops)
 109 
 110   # Use a set of update ops in addition to the default updates:
 111   tf.add_to_collection(tf.GraphKeys.UPDATE_OPS, my_update0)
 112   tf.add_to_collection(tf.GraphKeys.UPDATE_OPS, my_update1)
 113 
 114   train_op = tf.contrib.training.create_train_op(
 115      total_loss,
 116      optimizer)
 117 
 118   # Which is the same as:
 119   train_op = tf.contrib.training.create_train_op(
 120      total_loss,
 121      optimizer,
 122      update_ops=tf.get_collection(tf.GraphKeys.UPDATE_OPS))
 123 
 124 ******************************************
 125 * Initializing a model from a checkpoint *
 126 ******************************************
 127 
 128 It is common to want to 'warm-start' a model from a pre-trained checkpoint.
 129 One can use a tf.Scaffold and an initializing function to do so.
 130 
 131   ...
 132 
 133   # Create the train_op
 134   train_op = tf.contrib.training.create_train_op(total_loss, optimizer)
 135 
 136   # Create the initial assignment op
 137   checkpoint_path = '/path/to/old_model_checkpoint'
 138   variables_to_restore = tf.contrib.framework.get_model_variables()
 139   init_fn = tf.contrib.framework.assign_from_checkpoint_fn(
 140       checkpoint_path, variables_to_restore)
 141 
 142   # Run training.
 143   scaffold = tf.Scaffold(init_fn=init_fn)
 144   tf.contrib.training.train(train_op, my_log_dir, scaffold=scaffold)
 145 
 146 ***************************************************************************
 147 * Initializing a model from a checkpoint whose variable names don't match *
 148 ***************************************************************************
 149 
 150 At times, a user may want to initialize a new model with values from a
 151 checkpoint whose variable names do not match those of the current model. In this
 152 case, one needs to create a mapping from the checkpoint variable names to the
 153 current model variables. This requires only a small modification of the code
 154 above:
 155   ...
 156   # Creates a model with two variables, var0 and var1
 157   predictions = MyModel(images)
 158   ...
 159 
 160   # Create the train_op
 161   train_op = tf.contrib.training.create_train_op(total_loss, optimizer)
 162 
 163   checkpoint_path = '/path/to/old_model_checkpoint'
 164 
 165   # Create the mapping:
 166   variables_to_restore = {
 167       'name_var_0_in_checkpoint':
 168           tf.contrib.framework.get_unique_variable('var0'),
 169       'name_var_1_in_checkpoint':
 170           tf.contrib.framework.get_unique_variable('var1')
 171   }
 172   init_fn = tf.contrib.framework.assign_from_checkpoint_fn(
 173         checkpoint_path, variables_to_restore)
 174   scaffold = tf.Scaffold(init_fn=init_fn)
 175 
 176   # Run training.
 177   tf.contrib.training.train(train_op, my_log_dir, scaffold=scaffold)
 178 
 179 
 180 *************************************************
 181 * Fine-Tuning Part of a model from a checkpoint *
 182 *************************************************
 183 
 184 Rather than initializing all of the weights of a given model, we sometimes
 185 only want to restore some of the weights from a checkpoint. To do this, one
 186 need only filter those variables to initialize as follows:
 187 
 188   ...
 189 
 190   # Create the train_op
 191   train_op = tf.contrib.training.create_train_op(total_loss, optimizer)
 192 
 193   checkpoint_path = '/path/to/old_model_checkpoint'
 194 
 195   # Specify the variables to restore via a list of inclusion or exclusion
 196   # patterns:
 197   variables_to_restore = tf.contrib.framework.get_variables_to_restore(
 198       include=["conv"], exclude=["fc8", "fc9])
 199   # or
 200   variables_to_restore = tf.contrib.framework.get_variables_to_restore(
 201       exclude=["conv"])
 202 
 203   init_fn = tf.contrib.framework.assign_from_checkpoint_fn(
 204       checkpoint_path, variables_to_restore)
 205   scaffold = tf.Scaffold(init_fn=init_fn)
 206 
 207   # Run training.
 208   tf.contrib.training.train(train_op, my_log_dir, scaffold=scaffold)
 209 
 210 ******************************************************
 211 * Initializing model variables from values in memory *
 212 ******************************************************
 213 
 214 One may want to initialize the weights of a model from values coming from an
 215 arbitrary source (a text document, matlab file, etc). While this is technically
 216 feasible using assign operations, this strategy results in the values of your
 217 weights being stored in the graph. For large models, this becomes prohibitively
 218 large. However, it's possible to perform this initial assignment without having
 219 to store the values of the initial model in the graph itself by using
 220 placeholders and a feed dictionary:
 221 
 222   ...
 223 
 224   # Create the train_op
 225   train_op = tf.contrib.training.create_train_op(total_loss, optimizer)
 226 
 227   # Create the mapping from variable names to values:
 228   var0_initial_value = ReadFromDisk(...)
 229   var1_initial_value = ReadFromDisk(...)
 230 
 231   var_names_to_values = {
 232     'var0': var0_initial_value,
 233     'var1': var1_initial_value,
 234   }
 235 
 236   init_fn = tf.contrib.framework.assign_from_values_fn(var_names_to_values)
 237   scaffold = tf.Scaffold(init_fn=init_fn)
 238 
 239   # Run training.
 240   tf.contrib.training.train(train_op, my_log_dir, scaffold=scaffold)
 241 """

 271 def add_gradients_summaries(grads_and_vars):
 272   """Add summaries to gradients.
 273 
 274   Args:
 275     grads_and_vars: A list of gradient to variable pairs (tuples).
 276 
 277   Returns:
 278     The list of created summaries.
 279   """
 280   summaries = []
 281   for grad, var in grads_and_vars:
 282     if grad is not None:
 283       if isinstance(grad, ops.IndexedSlices):
 284         grad_values = grad.values
 285       else:
 286         grad_values = grad
 287       summaries.append(
 288           summary.histogram(var.op.name + '_gradient', grad_values))
 289       summaries.append(
 290           summary.scalar(var.op.name + '_gradient_norm',
 291                          clip_ops.global_norm([grad_values])))
 292     else:
 293       logging.info('Var %s has no gradient', var.op.name)
 294 
 295   return summaries

 298 def clip_gradient_norms(gradients_to_variables, max_norm):
 299   """Clips the gradients by the given value.
 300 
 301   Args:
 302     gradients_to_variables: A list of gradient to variable pairs (tuples).
 303     max_norm: the maximum norm value.
 304 
 305   Returns:
 306     A list of clipped gradient to variable pairs.
 307   """
 308   clipped_grads_and_vars = []
 309   for grad, var in gradients_to_variables:
 310     if grad is not None:
 311       if isinstance(grad, ops.IndexedSlices):
 312         tmp = clip_ops.clip_by_norm(grad.values, max_norm)
 313         grad = ops.IndexedSlices(tmp, grad.indices, grad.dense_shape)
 314       else:
 315         grad = clip_ops.clip_by_norm(grad, max_norm)
 316     clipped_grads_and_vars.append((grad, var))
 317   return clipped_grads_and_vars
 318 
 319 
 320 def clip_gradient_norms_fn(max_norm):
 321   """Returns a `transform_grads_fn` function for gradient clipping."""
 322   def clip_norms(gradients_to_variables):
 323     return clip_gradient_norms(gradients_to_variables, max_norm)
 324   return clip_norms

 327 def multiply_gradients(grads_and_vars, gradient_multipliers):
 328   """Multiply specified gradients.
 329 
 330   Args:
 331     grads_and_vars: A list of gradient to variable pairs (tuples).
 332     gradient_multipliers: A map from either `Variables` or `Variable` op names
 333       to the coefficient by which the associated gradient should be scaled.
 334 
 335   Returns:
 336     The updated list of gradient to variable pairs.
 337 
 338   Raises:
 339     ValueError: If `grads_and_vars` is not a list or if `gradient_multipliers`
 340     is empty or None or if `gradient_multipliers` is not a dictionary.
 341   """
 342   if not isinstance(grads_and_vars, list):
 343     raise ValueError('`grads_and_vars` must be a list.')
 344   if not gradient_multipliers:
 345     raise ValueError('`gradient_multipliers` is empty.')
 346   if not isinstance(gradient_multipliers, dict):
 347     raise ValueError('`gradient_multipliers` must be a dict.')
 348 
 349   multiplied_grads_and_vars = []
 350   for grad, var in grads_and_vars:
 351     if var in gradient_multipliers or var.op.name in gradient_multipliers:
 352       key = var if var in gradient_multipliers else var.op.name
 353       if grad is None:
 354         raise ValueError('Requested multiple of `None` gradient.')
 355 
 356       if isinstance(grad, ops.IndexedSlices):
 357         tmp = grad.values * constant_op.constant(
 358             gradient_multipliers[key], dtype=grad.dtype)
 359         grad = ops.IndexedSlices(tmp, grad.indices, grad.dense_shape)
 360       else:
 361         grad *= constant_op.constant(
 362             gradient_multipliers[key], dtype=grad.dtype)
 363     multiplied_grads_and_vars.append((grad, var))
 364   return multiplied_grads_and_vars

 370 def create_train_op(total_loss,
 371                     optimizer,
 372                     global_step=_USE_GLOBAL_STEP,
 373                     update_ops=None,
 374                     variables_to_train=None,
 375                     transform_grads_fn=None,
 376                     summarize_gradients=False,
 377                     gate_gradients=tf_optimizer.Optimizer.GATE_OP,
 378                     aggregation_method=None,
 379                     colocate_gradients_with_ops=False,
 380                     check_numerics=True):
 381   """Creates an `Operation` that evaluates the gradients and returns the loss.
 382 
 383   Args:
 384     total_loss: A `Tensor` representing the total loss.
 385     optimizer: A tf.Optimizer to use for computing the gradients.
 386     global_step: A `Tensor` representing the global step variable. If left as
 387       `_USE_GLOBAL_STEP`, then tf.contrib.framework.global_step() is used.
 388     update_ops: An optional list of updates to execute. If `update_ops` is
 389       `None`, then the update ops are set to the contents of the
 390       `tf.GraphKeys.UPDATE_OPS` collection. If `update_ops` is not `None`, but
 391       it doesn't contain all of the update ops in `tf.GraphKeys.UPDATE_OPS`,
 392       a warning will be displayed.
 393     variables_to_train: an optional list of variables to train. If None, it will
 394       default to all tf.trainable_variables().
 395     transform_grads_fn: A function which takes a single argument, a list of
 396       gradient to variable pairs (tuples), performs any requested gradient
 397       updates, such as gradient clipping or multipliers, and returns the updated
 398       list.
 399     summarize_gradients: Whether or not add summaries for each gradient.
 400     gate_gradients: How to gate the computation of gradients. See tf.Optimizer.
 401     aggregation_method: Specifies the method used to combine gradient terms.
 402       Valid values are defined in the class `AggregationMethod`.
 403     colocate_gradients_with_ops: Whether or not to try colocating the gradients
 404       with the ops that generated them.
 405     check_numerics: Whether or not we apply check_numerics.
 406 
 407   Returns:
 408     A `Tensor` that when evaluated, computes the gradients and returns the total
 409       loss value.
 410   """
 411   if global_step is _USE_GLOBAL_STEP:
 412     global_step = training_util.get_or_create_global_step()
 413 
 414   # Update ops use GraphKeys.UPDATE_OPS collection if update_ops is None.
 415   global_update_ops = set(ops.get_collection(ops.GraphKeys.UPDATE_OPS))
 416   if update_ops is None:
 417     update_ops = global_update_ops
 418   else:
 419     update_ops = set(update_ops)
 420   if not global_update_ops.issubset(update_ops):
 421     logging.warning('update_ops in create_train_op does not contain all the '
 422                     ' update_ops in GraphKeys.UPDATE_OPS')
 423 
 424   # Make sure update_ops are computed before total_loss.
 425   if update_ops:
 426     with ops.control_dependencies(update_ops):
 427       barrier = control_flow_ops.no_op(name='update_barrier')
 428     total_loss = control_flow_ops.with_dependencies([barrier], total_loss)
 429 
 430   if variables_to_train is None:
 431     # Default to tf.trainable_variables()
 432     variables_to_train = tf_variables.trainable_variables()
 433   else:
 434     # Make sure that variables_to_train are in tf.trainable_variables()
 435     for v in variables_to_train:
 436       assert v in tf_variables.trainable_variables()
 437 
 438   assert variables_to_train
 439 
 440   # Create the gradients. Note that apply_gradients adds the gradient
 441   # computation to the current graph.
 442   grads = optimizer.compute_gradients(
 443       total_loss,
 444       variables_to_train,
 445       gate_gradients=gate_gradients,
 446       aggregation_method=aggregation_method,
 447       colocate_gradients_with_ops=colocate_gradients_with_ops)
 448 
 449   if transform_grads_fn:
 450     grads = transform_grads_fn(grads)
 451 
 452   # Summarize gradients.
 453   if summarize_gradients:
 454     with ops.name_scope('summarize_grads'):
 455       add_gradients_summaries(grads)
 456 
 457   # Create gradient updates.
 458   grad_updates = optimizer.apply_gradients(grads, global_step=global_step)
 459 
 460   with ops.name_scope('train_op'):
 461     # Make sure total_loss is valid.
 462     if check_numerics:
 463       total_loss = array_ops.check_numerics(total_loss,
 464                                             'LossTensor is inf or nan')
 465 
 466     # Ensure the train_tensor computes grad_updates.
 467     train_op = control_flow_ops.with_dependencies([grad_updates], total_loss)
 468 
 469   # Add the operation used for training to the 'train_op' collection
 470   train_ops = ops.get_collection_ref(ops.GraphKeys.TRAIN_OP)
 471   if train_op not in train_ops:
 472     train_ops.append(train_op)
 473 
 474   return train_op

 477 def train(train_op,
 478           logdir,
 479           master='',
 480           is_chief=True,
 481           scaffold=None,
 482           hooks=None,
 483           chief_only_hooks=None,
 484           save_checkpoint_secs=600,
 485           save_summaries_steps=100,
 486           config=None,
 487           max_wait_secs=7200,
 488           run_metadata=None):
 489   """Runs the training loop.
 490 
 491   Args:
 492     train_op: A `Tensor` that, when executed, will apply the gradients and
 493       return the loss value.
 494     logdir: The directory where the graph and checkpoints are saved.
 495     master: The URL of the master.
 496     is_chief: Specifies whether or not the training is being run by the primary
 497       replica during replica training.
 498     scaffold: An tf.train.Scaffold instance.
 499     hooks: List of `tf.train.SessionRunHook` callbacks which are run inside the
 500       training loop.
 501     chief_only_hooks: List of `tf.train.SessionRunHook` instances which are run
 502       inside the training loop for the chief trainer only.
 503     save_checkpoint_secs: The frequency, in seconds, that a checkpoint is saved
 504       using a default checkpoint saver. If `save_checkpoint_secs` is set to
 505       `None`, then the default checkpoint saver isn't used.
 506     save_summaries_steps: The frequency, in number of global steps, that the
 507       summaries are written to disk using a default summary saver. If
 508       `save_summaries_steps` is set to `None`, then the default summary saver
 509       isn't used.
 510     config: An instance of `tf.ConfigProto`.
 511     max_wait_secs: Maximum time workers should wait for the session to
 512       become available. This should be kept relatively short to help detect
 513       incorrect code, but sometimes may need to be increased if the chief takes
 514       a while to start up.
 515     run_metadata: A [`RunMetadata`] protocol buffer.
 516 
 517   Returns:
 518     the value of the loss function after training.
 519 
 520   Raises:
 521     ValueError: if `logdir` is `None` and either `save_checkpoint_secs` or
 522     `save_summaries_steps` are `None.
 523   """
 524   if logdir is None and is_chief:
 525     if save_summaries_steps:
 526       raise ValueError(
 527           'logdir cannot be None when save_summaries_steps is not None')
 528 
 529     if save_checkpoint_secs:
 530       raise ValueError(
 531           'logdir cannot be None when save_checkpoint_secs is not None')
 532 
 533   with monitored_session.MonitoredTrainingSession(
 534       master=master,
 535       is_chief=is_chief,
 536       checkpoint_dir=logdir,
 537       scaffold=scaffold,
 538       hooks=hooks,
 539       chief_only_hooks=chief_only_hooks,
 540       save_checkpoint_secs=save_checkpoint_secs,
 541       save_summaries_steps=save_summaries_steps,
 542       config=config,
 543       max_wait_secs=max_wait_secs) as session:
 544     loss = None
 545     while not session.should_stop():
 546       loss = session.run(train_op, run_metadata=run_metadata)
 547   return loss
